== Description ==
Ads is a plugin that allows you to insert ads anywhere into your post just by using a simple shortcode. Most importantly there have no limits. You can create hundreds of Ad codes by your own or can get from different ad networks like Google AdSense or Bing Ads. There is lot more options. You can select the alignment for your ads. By using this plugin you cannot only insert Ads to the post, this plugin will give you ability to place any JavaScript or HTML codes into post.

= Features =

1. Easy to use Google AdSense and Bing Ads Code

2. You can use any JavaScript or HTML codes

3. You can use different Ad Networks' code at a time

4. No limite to create your Ads into your Website or Blog

See <a href="http://appstore.probashitimes.com/" target="_blank">Documentation</a>  


== Installation ==

1. Download the zip file and extract the contents.

2. Upload the 'ads' folder to your plugins directory (wp-content/plugins/).

3. Activate the plugin through the 'Plugins' menu in WordPress.


= How to Config =

1. Go to ‘Settings -> ADs’

2. Click on ADD New to add new Ad

3. Fill the form with your data correctly and select Align type.

4. Save form.

= Placing Ad into a Post =

1. You need to use shortcode for this [metaads ad=your_ad_name]

2. This shortcode have an attribute named “ad” you need to place your Ad name there.

3. For Example : – If your Ad name is “MyAds1” then your shortcode will be like [metaads ad=MyAds1]

4. Just place your shortcode where you need to display your Ad.


= How to modifying ADs = 

1. Go to ‘Settings -> ADs’

2. Select an Ad name you need to modify from the drop down list

3. Click on Modify button

4. Modify the data and click on Update button

5. Click on Delete button if you need to remove your Ad
